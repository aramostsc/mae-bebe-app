# User Stories

## Mãe no primeiro filho

- Como mãe pela primeira vez, quero abrir a app e sentir que há uma orientação calma, para não me sentir sozinha nas dúvidas do dia.
- Como mãe insegura, quero receber dicas simples e não absolutas, para me sentir apoiada sem sentir julgamento.
- Como mãe cansada, quero encontrar rapidamente o que preciso, para a app não se tornar mais uma tarefa.
- Como mãe emocionalmente sobrecarregada, quero uma experiência serena, para sentir que posso respirar e reorganizar-me.

## Pós-parto

- Como mãe no pós-parto, quero saber que tipo de movimento é adequado à minha fase, para retomar o corpo com cuidado.
- Como mãe com sintomas desconfortáveis, quero avisos claros de segurança, para saber quando devo parar e procurar ajuda profissional.
- Como mãe com pouca energia, quero sugestões simples de alimentação e hidratação, para cuidar de mim sem planos rígidos.

## Bebé

- Como mãe de um bebé pequeno, quero ver a idade atual em semanas ou meses, para acompanhar melhor a fase em que estamos.
- Como mãe de um bebé em introdução alimentar, quero orientação por idade, para ter uma base simples antes de falar com o pediatra.
- Como mãe atenta ao desenvolvimento, quero dicas por categoria, para ter pequenas ideias úteis sem ficar assoberbada.

## Organização

- Como mãe com consultas e vacinas para gerir, quero uma agenda simples, para não depender da memória.
- Como mãe que valoriza pequenos marcos, quero registar eventos e memórias, para acompanhar o crescimento do bebé.
- Como família, queremos no futuro poder partilhar lembretes e responsabilidades, para que a carga não esteja só numa pessoa.

## Memórias

- Como mãe, quero guardar uma foto com uma nota curta, para registar momentos importantes sem esforço.
- Como mãe, quero ver as fotos organizadas pela idade do bebé, para reviver a evolução de forma natural.

## Produto pessoal primeiro

- Como criador da app, quero que a primeira versão seja útil para uso real pela minha mulher, para validar valor antes de pensar em app pública ou negócio.
- Como criador da app, quero manter as decisões documentadas, para o produto evoluir com intenção e não apenas por acumulação de funcionalidades.
